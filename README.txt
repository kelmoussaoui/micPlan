========================================
        micPlan - Application Portable
========================================

DESCRIPTION
-----------
micPlan est une application de planification des postes de laboratoire
developpee en Python avec une interface web Streamlit.

INSTALLATION ET UTILISATION
---------------------------
1. COPIEZ ce dossier complet sur votre disque reseau ou PC Windows
2. Double-cliquez sur "Lancer_micPlan.bat" pour demarrer l'application
3. L'application s'ouvrira automatiquement dans votre navigateur par defaut
4. Pour arreter l'application, fermez la fenetre de commande

FONCTIONNALITES
---------------
- Gestion des postes et configurations
- Planification des equipes
- Base de donnees SQLite portable
- Interface web moderne et intuitive
- Gestion des utilisateurs et authentification

REQUIS SYSTEME
--------------
- Windows 7/8/10/11 (32-bit ou 64-bit)
- Navigateur web moderne (Chrome, Firefox, Edge)
- Pas d'installation Python requise
- Pas de droits administrateur necessaires

STRUCTURE DES DONNEES
---------------------
- Base de donnees SQLite integree dans le dossier "data/"
- Sauvegardes automatiques dans "data/backups/"
- Toutes les donnees sont locales a ce dossier

DEPANNAGE
----------
- Si l'application ne se lance pas, verifiez que le dossier est complet
- Assurez-vous qu'aucun antivirus ne bloque l'execution
- Redemarrez l'application si necessaire
